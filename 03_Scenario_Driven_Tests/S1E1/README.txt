1.环境要求：
二台小站（教师端与学生端）
教师端：1个HDMI摄像头（HDMI_CAM_1）,1个IPC摄像头，本地管理页面算法必须配置好，分别为老师识别HDMI和学生识别IPC
学生端：1个HDMI摄像头（默认CAM_0口）
显示器要求：分辨率1920*1080

2.脚本执行方式为:
python 脚本名.py  

3.执行步骤：
step1:    教师端: （使用方式）: 
             SSH 当前小站 终端输入 python 4F-4.1D-4.2T-4.3S-4.4RA.py
             参数化构建  -ip=10.12.224.135
                       -TCam=HDMI   指定教师摄像头类型，本地管理需要手动配置教师算法
                       -SCam=IPC    指定学生摄像头类型，本地管理需要手动配置学生算法
             开启四宫格，1宫导播不带录制，2宫为教师摄像头回显，3宫为学生摄像头回显，4宫远端声网拉流

step2:   学生端：
             SSH 另一台小站 终端输入 python student_create_remote.py  （-ip=10.12.224.135）
		     参数化构建 -ip指定推流的小站，缺省值为当前小站
             学生端声网推流，没有回显。

step:双端通用：
             python delete_all_tasks_rtc.py （-ip=10.12.224.135）参数化构建，可跟ip参数，指定删除对应小站的任务，不跟参数默认删除本台小站
结束当前的任务
