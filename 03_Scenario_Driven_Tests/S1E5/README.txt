1. 环境要求
教师端：2个HDMI摄像头（HDMI_CAM_0、CAM_1），1个IPC网络摄像头用于拉流
显示器要求：分辨率1920*1080

2. 脚本执行方式为:
python 脚本名.py  

教师端:
SSH 当前小站 （测试方式可选）
（1）终端输入  python Act_Story.py
        进行故事动画演绎，动画流程如下：
        ①创建三宫格窗口，1宫为导播带录制声网推流，2宫为教师摄像头回显，3宫为学生摄像头回显
        ②创建第四个宫格窗口，该窗口为IPC网络摄像头拉流
        ③删除第①步创建的三宫格窗口
        ④对第四宫格窗口进行顺时针与逆时针各三圈的宫格位置变换，之后回归原位
        ⑤重新执行第①步，创建回前三宫窗口
        ⑥删除所有窗口
        至此动画结束

（2）也可以执行单功能脚本，做各种动作
        ① python 4F-4.1D-4.2T-4.3S.py
        创建三宫格窗口，1宫为导播带录制声网推流，2宫为教师摄像头回显，3宫为学生摄像头回显，有独立TaskId
        ②python create_remote_IPC.py
        创建第四宫IPC网络摄像头拉流窗口，有独立TaskId
        ③python delete_task_by_action_name.py   (-n=4F-4.1D-4.2T-4.3S)
        参数化构建，可以删除某一个动作任务，可跟参数有4F-4.1D-4.2T-4.3S与create_remote_IPC
        删除对应动作的任务，不跟参数，默认为删除第①步执行的动作任务 创建的窗口
        ④python change_geometry.py （-c=0,0,960,540）
        参数化构建，指定第四宫格窗口进行位置切换，前两个数0，0 为宫格窗口左上角距离显示器左上角（x，y）偏移坐标，后两个数（960，540）为该窗口的分辨率。
        ⑤python change_geometry_animation.py
         由上述脚本衍生的脚本，对第四宫格窗口进行顺逆时针三圈变换
        ⑥python delete_all_tasks.py
         删除当前创建的所有任务
