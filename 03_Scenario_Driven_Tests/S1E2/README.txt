1. 环境要求
教师端：
2个HDMI摄像头（HDMI_CAM_0、CAM_1口），需要本地管理配置老师识别和学生识别，一个IPC网络摄像头用于拉流，
显示器要求：分辨率1920*1080

2. 脚本执行方式为:
python 脚本名.py  

教师端:
SSH 当前小站 终端输入 python P2-4F-4.1DR-4.2T-4.3S-4.4RC.py
开启四宫格，1宫导播带录制，2宫为教师摄像头回显，3宫为学生摄像头回显，4宫远端IPC网络摄像头拉流
             参数化构建  -ip=10.12.224.135
                       -TCam=HDMI   指定教师摄像头类型，本地管理需要手动配置教师算法  （可选参数）
                       -SCam=IPC    指定学生摄像头类型，本地管理需要手动配置学生算法 （可选参数）
                       -rtsp=rtsp://10.12.224.57/ch2  指定IPC网络摄像头拉流的地址 （可选参数）
             EX： python P2-4F-4.1DR-4.2T-4.3S-4.4RC.py -ip=10.12.224.135 -TCam=HDMI  -SCam=IPC -rtsp=rtsp://10.12.224.57/ch2
