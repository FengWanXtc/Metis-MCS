1. 环境要求
1台小站
教师端：本地管理1个HDMI配老师

显示器要求：分辨率1920*1080

2. 脚本执行方式

方式：python Record_Size.py  根据字节大小切分录制文件  -ip 指定对应IP小站，缺省值为当前小站
                                                 -u: 分块大小，默认1000KB
python Record_Time.py  根据字节大小切分录制文件  -ip 指定对应IP小站，缺省值为当前小站
                                                 -u: 分块大小，默认10s
