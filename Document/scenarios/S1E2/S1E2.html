4.2.1 外设配置

  • 1台Metis Station小站作为主讲端

  • 2个摄像头需要本地管理配置老师识别和学生识别， 加上 1个网络摄像头作为拉流回显。

  • 显示器要求：分辨率1920*1080及以上

4.2.2. DemoMCS 演示

  • 说明：

    1. 该MCS的操作与S1E1的操作相仿， 脚本调用相应接口自动填充VideoSpecs中的DeviceId字段，形成原始视频流，经过VideoAiSpecs 形成算法视频流。随后将导播一路Spec VideoDirectorSpec的Name字段 作为VideoCodecSpecs 中SourceName字段视频源， 编码视频流。

    2. 导播路视频流经过编码后通过 RecordSpecs 进行录制，该Spec中的VideoCodecName字段值就是经过编码的流，即VideoCodecSpecs 的Name字段。Path字段是录制的视频文件存放的路径

    3. CodecStreamSpecs是拉流Spec，支持拉取RTSP,RTMP,TCP视频流，其URI字段填写正确的地址，即可拉取到远端视频流，在 本例中通过RenderSpecs将拉取到的远端视频流回显出来。

    4. 各个Spec字段的详细描述请阅读 MA API 文档。
