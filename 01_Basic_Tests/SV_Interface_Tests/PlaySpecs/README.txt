1. 环境要求
HDMI 3个输入口皆接上外设，本地管理将相应视频输入设备打开，并且添加一个IPC摄像头为打开状态
接上一个HDMI显示器1920*1080

一个有线MIC插在MIC 1 口


2. 脚本执行方式为:
python 脚本名.py

脚本说明：
结束任务通过 delete_all_tasks.py -ip=XXXXXXX结束任务

3. 场景测试内容:
    01: 对有线MIC，PC播放音乐，只有 MIC 的声音能从扬声器里放出来
    02: 同上，只有 PC 播放的音乐能从扬声器里播放出来
    03: 对无线MIC说话，PC播放音乐，只有 无线MIC 的声音能从扬声器里放出来
    （版本更迭，无线MIC和有线MIC共用一个DeviceId）
    04：对无线MIC和有线MIC说话，均能从扬声器放出声音
    05: 对有线和无线MIC说话，PC播放音乐，三者都能从扬声器里播放出来
    06：对有线MIC说话，PC播放音乐，两者声音从扬声器里播放出来
    07：增大PC的输出音量
    08：Part 1 修改MIC的输入音量为0，只有PC的声音可以从扬声器出来
        Part 2 将音量改回70，MIC声音可以从扬声器出来，不影响PC输出
    09：Error 源 测试，返回码400。
    10：Part 1 修改MIC输入音量为1000，设定超过100即是100，MIC声音可以从扬声器出来
        Part 2 修改PC输入音量为-100，低于0即是0，PC音量无法传出
